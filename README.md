# Falco operator

A [Juju](https://juju.is/) charm deploying and managing [Falco](https://falco.org/), an open-source
runtime security tool for physical and virtual machines. This subordinate charm simplifies the
deployment, configuration, and lifecycle management of Falco instances.

Like any Juju charm, this charm supports one-line deployment, configuration, integration, and more.

For information about how to deploy, integrate, and manage this charm, see the official [Falco Operator Documentation](https://charmhub.io/falco/docs).
